🎯 실무에서 사용하는 대표 전략
1️⃣ 검색량(조회수) 기반 필터링
monthlyPcQcCnt + monthlyMobileQcCnt (PC+모바일 합산 조회수)

일정 기준 이하 (예: 500 미만 or 1000 미만)은 제외하는 방식

2️⃣ 클릭률(CTR) 기반 필터링
monthlyAvePcCtr & monthlyAveMobileCtr 사용

일반적으로 CTR이 너무 낮은 키워드는 제외 (ex: 1% 미만 필터링)

3️⃣ 경쟁도(compIdx) 필터링
'높음', '중간', '낮음' 으로 나뉘는 값

'높음'은 너무 경쟁이 치열할 수 있어,
'중간' & '낮음' 위주로 선택하는 경우가 많음

🧩 내가 추천하는 필터링 플로우
조회수 합산 기준 필터링
(PC + 모바일 합산 조회수)

monthlyPcQcCnt + monthlyMobileQcCnt ≥ 500

CTR 기준 필터링

monthlyAveMobileCtr ≥ 1.0 or monthlyAvePcCtr ≥ 1.0

경쟁도 필터

compIdx가 '중간' or '낮음' 만 선택

이후 상위 CTR 순으로 정렬 후 max_tags 만큼 선택

⚠ 주의할 점
CTR만 높고 조회수가 낮은 키워드는 "니치"일 수 있음 (이걸 롱테일로 쓸 수도 있음)

경쟁도가 '높음'인 키워드도 혼합형 조합용으로는 일부 활용 가능

🎯 요약하면
"조회수 + CTR + 경쟁도"를 복합적으로 고려
→ 조회수 & CTR 높은 키워드 위주로 뽑고,
→ 니치 키워드는 롱테일 조합용으로 활용

💡 실무 예시
relKeyword	합산 조회수	모바일 CTR	경쟁도	채택 여부
실리콘테이프	1500	2.3%	중간	✅
양면테이프 추천	600	0.9%	낮음	❌ (CTR 낮음)
폼테이프	1200	3.0%	중간	✅
강력테이프	400	1.5%	낮음	❌ (조회수 낮음)


### 그렇다면 이중 연관도가 높다는것은??????

🎯 "연관도"를 판단할 주요 지표
1️⃣ monthlyPcQcCnt + monthlyMobileQcCnt (조회수 합산)
조회수가 많다는 건, 메인키워드와 함께 사용자들이 자주 검색한 키워드라는 의미입니다.

검색량이 적은 키워드는 연관도가 낮을 확률이 큼

2️⃣ CTR (monthlyAvePcCtr + monthlyAveMobileCtr)
조회수는 많지만 클릭률이 낮은 키워드는 "낚시성 키워드"일 수 있음

CTR이 높은 키워드는 메인 키워드와 의도적으로 연관성이 높을 확률이 높음

3️⃣ plAvgDepth (평균 검색 심도)
네이버에서는 이걸 사용자가 몇 단계까지 검색했는지로 표시

일반적으로 깊이가 낮을수록 연관성이 높음

예: plAvgDepth ≤ 3~5 → 메인키워드와 바로 연관된 키워드

깊이가 높음 (6 이상) → 보통 메인 키워드와는 덜 관련된 "후속 탐색" 키워드일 가능성

4️⃣ compIdx (경쟁도)
경쟁도가 높다고 해서 반드시 연관도가 높다고 보긴 어려움

다만 연관성 필터에는 참고값으로만 활용 가능 (서브 조건)

🧠 실전 연관도 필터링 추천
조회수 합산 ≥ 1000

CTR ≥ 1% 이상

검색 깊이(plAvgDepth) ≤ 5
→ 이렇게 3가지 조건을 걸면 실제 연관도가 높은 키워드가 잘 뽑힙니다.

🔵 실제 현업 사용법
조회수 + CTR + 검색깊이 조합을 연관도 필터로 가장 많이 사용

이걸 1차 필터로 걸고, 이후 경쟁도나 롱테일 여부를 보고 판단

필요하면 이걸 그대로 자동화 로직으로 바로 만들어줄까요?
예: filter_related_keywords_by_score() 같은 함수로요 🚀


🎯 마켓 상황 및 최적화 전략
🟢 현재 마켓 상황
**일반상품명(기본상품명)**으로 낮은 마진을 설정하여 **판매량(매출 규모)**을 우선적으로 확보하는 구조

**가공상품명(프리미엄 상품명)**으로 높은 마진을 챙겨, 이익률 확보를 병행하는 구조

볼륨형(낮은마진 + 대량판매) + 마진형(고마진 + 차별화) 이중 트랙 전략 사용 중

🧩 최적화 전략
✅ 1. 일반상품명 전략
상품명: 혼합형 (메인 키워드 + 빈도우위 키워드 조합)
→ ex: 양면테이프 초강력 투명 실리콘폼

태그: 조회수 기반 연관검색어 태그
→ 네이버 SEO 기준으로 상위 노출 및 대중적 유입량 확보에 중점

목표:

유입 트래픽 극대화

대중적 키워드를 통한 볼륨형 매출 증가

✅ 2. 가공상품명 전략
상품명: 롱테일형 (메인 키워드 + 틈새/하위 키워드 조합)
→ ex: 양면테이프 찍찍이 방수테이프 부직포

태그: CTR + 조회수 기반의 하이브리드 태그

CTR 우선 + 조회수 보조로 혼합형 태그 구성

예: CTR 5개 + 조회수 5개 (또는 6:4 비율로 커스터마이징 가능)

목표:

구매 의도가 명확한 고객 타겟팅

고마진 상품의 전환율 극대화

🧠 전략의 핵심
혼합형 상품명 + 대중적인 태그 → SEO 노출 + 대량 유입 확보

롱테일형 상품명 + 전환형 태그 → 틈새 시장 공략 + 고전환 확보

→ 두 전략이 유입과 전환을 상호 보완하여 마켓의 "안정적인 볼륨 확보 + 수익성 개선"에 기여

🚩 AB테스트 계획
혼합형 + 조회수 태그 세트 vs 롱테일형 + CTR+조회수 하이브리드 태그 세트를 실전 마켓에 적용 후

노출수, 클릭수, 전환률 데이터를 수집하여 효율 분석

✨ 추가 제안
상황에 맞춰 AB테스트 결과를 주기적으로 반영하여
혼합형 전략과 롱테일형 전략을 탄력적으로 운영
