# 📌 상품명 가공 자동화를 위한 전략 정리

---

## 1. 🎯 메인키워드 추출 기준

### ✅ 기본원칙  
> "상품명에 포함된 단어 중, 해당 상품의 **주요 기능/카테고리/제품군을 대표하는 단어**를 메인키워드로 설정한다."

### ✅ 추출 우선순위

| 우선순위 | 방식 | 설명 | 오류 가능성 |
|----------|------|------|-------------|
| 1순위 | **카테고리 기반 매칭** | 상품명에 카테고리 단어가 존재하면 이를 메인키워드로 사용 | 낮음 |
| 2순위 | **상위판매자 분석** | 유사상품 상위 10~15개 상품명을 분석하여 가장 많이 등장하는 단어를 추출 | 중간 |
| 3순위 | **GPT 제안 또는 규칙기반 대체** | 위 방법이 불가능한 경우 GPT 또는 사전 등록된 규칙 기반 추출 | 높음 |

### ✅ 예시

- `기본상품명`: "틈새청소솔 블랙"
  - 카테고리 기반: `청소솔`
  - 상위판매자 중복 키워드: `청소솔`, `브러쉬`
  - GPT 추출: `틈새`

→ **최종 메인키워드: `청소솔`**

---

## 2. ⚙️ 자동화 흐름 설계

### ✅ 자동화 전체 흐름 요약

[기본상품명]
↓
[메인키워드 추출]
↓
[연관검색어 + 상위 키워드 분석]
↓
[브랜드 키워드 필터링 + 키프리스 필터링]
↓
[고정 키워드 + 메인 키워드 + 유효 키워드 조합]
↓
[상품명 가공 완료]



### ✅ 자동화 단계별 설명

1. **메인키워드 추출**  
   - 카테고리 기반 우선
   - 실패 시 상위판매자 분석 → GPT fallback

2. **연관 키워드 분석**  
   - 네이버 연관검색어 API 및 상위상품 타이틀 분석

3. **브랜드 & 키프리스 필터링**  
   - 브랜드명 제거
   - 키프리스 등록 키워드 제거

4. **고정 키워드 결합**  
   - 상품명에서 메인 키워드를 제외한 단어들을 고정 키워드로 사용하여 조합

## 3. 🛠️ 수동 최적화 연동 전략

### ✅ 최적화 기준
> “판매 실적이 있거나 리뷰/전환률이 높아질 가능성이 있는 상품만 수동으로 정밀 최적화한다.”

### ✅ 수동 최적화 시 저장할 항목

| 항목 | 목적 |
|------|------|
| 메인 키워드 | 이후 자동화 기준 개선 |
| 사용한 카테고리 | 카테고리-키워드 매핑 데이터 누적 |
| 선택한 태그 | 검색태그 자동화에 사용 |
| 썸네일 패턴 | 클릭률 개선 참고 |

→ `dictionary[main_keyword]` 구조에 반영하여 자동화 품질 향상

---

## 4. 🧠 장기 자동화 전략 요약

| 단계 | 전략 |
|------|------|
| 초기 (지금) | GPT 기반 사전 + 카테고리 매칭으로 자동화 / 팔린 상품 수동 최적화 |
| 중기 (3~6개월) | 수동 최적화된 메인키워드/태그/카테고리 누적 → 자동화 보정 |
| 장기 (6개월~1년) | 대부분의 주요 상품군이 수동으로 보정됨 → 자동화 정확도 80% 이상 도달 → 자동화 중심 운영 가능 |
