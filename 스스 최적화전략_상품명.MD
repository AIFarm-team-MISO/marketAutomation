📦 스마트스토어 대량등록 위탁 판매자를 위한 최적화 전략 문서

---

## 🎯 전략 목표
- 대량등록 위탁모델에 적합한 상품명/검색태그 자동화 전략 수립
- 광고비 최소화, 유입과 전환의 극대화
- 반복 가능한 수작업 루틴을 자동화 기반으로 정형화

---

## 🧩 현재 판매 전략 개요
| 항목 | 설명 |
|------|------|
| 상품 수 | 1~3만개 등록 중 |
| 가격 정책 | 최소단가 판매 → 판매 발생 시 10% 인상 |
| 최적화 | 팔린 상품 대상: 상품명 + 태그 + 광고 설정 진행 |
| 광고 단가 | 기본 50원, 최적화 상품군은 80원 설정 |
| 전환 목표 | 순수 검색 유입 + 상품명 기반 키워드 광고 유입 |

---

## 🧠 통찰 기반 전략 방향성

### ✅ 핵심 통찰
| 통찰 번호 | 요점 |
|------------|------|
| 💡 1 | 상위판매자 상품명 키워드는 실제 경쟁강도/전환 고려의 결과일 수 있음 |
| 💡 2 | 연관검색어 API 결과는 필터링 후, 경쟁강도 분석에 유용하게 활용 가능 |
| 💡 3 | CTR과 조회수만으로 키워드 채택 여부를 판단하기엔 한계가 있음 → 경쟁강도 추가 분석 필요 |
| 💡 4 | 리치키워드(CTR↑ + 조회수↓)는 광고 없이도 유입 가능성이 높음 |
| 💡 5 | 빈도우위 키워드는 경쟁강도 낮거나 중간일 경우에만 활용 가치가 높음 |
| 💡 6 | 상품명은 자동화할 수 있지만, 광고는 자동화가 어려우므로 **상품명 중심 전략이 핵심** |

---

## 🧬 키워드 조합 전략

### [1] 이상적인 상품명 구조
```
[메인 키워드] + [리치 키워드] + [보완 키워드] + [고정 키워드]
```

- **메인 키워드**: 상품군을 대표 (예: 양면테이프)
- **리치 키워드**: CTR↑ & 조회수↓ & 경쟁강도↓ (광고 없이도 유입 기대)
- **보완 키워드**: 빈도↑ & 경쟁강도 중간 이하 (광고 유입 보조 역할)
- **고정 키워드**: 옵션/세트 정보 등 필수 정보

---

## 🛠 데이터 기반 상품명/태그 설계 기준
| 항목 | 내용 |
|-------|------|
| CTR | 1.5% 이상 선호 (단, 조회수 매우 낮으면 제외 가능) |
| 조회수 | 500~5000 선호 (너무 높으면 경쟁↑, 너무 낮으면 노출↓) |
| 경쟁강도 | 낮음 또는 중간 → 필터 우선 기준 |
| 평균 검색 깊이 | 1~5 선호 (너무 깊으면 연관도 낮음) |

---

## 🔁 최적화 루틴 유지 전략
- 오전: 주문처리 + 순환작업 + 최적화
- 오후: 광고 확인 + 추가 최적화 작업
- 하루 10~20개 최적화 → 6개월 이상 누적 시 데이터셋 형성 가능
- 최적화 상품명과 태그는 엑셀에 누적 저장 (향후 자동화 2.0 기반)

---

## 🚀 장기적 자동화 설계 흐름

1. 최적화 결과 데이터 누적 (엑셀)
2. 키워드 빈도 및 경쟁강도 자동 분석
3. 카테고리-키워드 매칭 평가 로직 개발
4. 상품명 자동 추천 알고리즘 구현
5. 검색태그 자동 추천 엔진 구축
6. 최종: 내가 만든 데이터로 나만의 추천 AI 구성

## 📝 결론
> 지금의 수작업은 단순한 노동이 아니라, **나만의 데이터셋을 학습시키는 정답 수집 과정**이다.

광고단가를 올릴 수 없다면 상품명을 바꾸고,
수작업으로 얻은 인사이트를 자동화로 확장하면,
그 누구도 모방할 수 없는 **나만의 경쟁력**을 만들 수 있다.

✅ CTR 높은 키워드 (리치 키워드) → 상품명 + 태그에 활용	광고 없이도 검색 전환 가능
✅ 빈도 키워드 + 경쟁 낮음 → 상품명에 활용 + 광고 ON	낮은 단가로도 광고 효과 가능
⚠️ 빈도 키워드 + 경쟁 높음 → 태그나 상품명에는 OK / 광고는 보류	광고해도 효과 없고 노출 불가 가능성


🧠 그래서 이상적인 키워드 조합 전략은?
구성 요소	설명
🎯 메인 키워드	상품군 대표 키워드 (예: 양면테이프)
🧬 리치 키워드	CTR↑ + 빈도↓ (→ 순수 검색 유입 가능성 높음)

### 더 최선을 원한다면 
✅ 상품명 구성 전략: 리치 → 보완 → 고정
순서	키워드 유형	설명	목적
①	🧬 리치 키워드	CTR↑ + 경쟁강도↓ + 카테고리 일치	순수 검색 유입 최적화
②	🧲 보완 키워드	빈도↑ + 경쟁강도↓	노출량 확보 + 광고 보완
③	🔒 고정 키워드	세트/옵션/사이즈 등 필수 정보	구매 결정 정보 제공
📌 이유 1. 네이버 상품명 검색 알고리즘은 앞쪽 키워드에 가중치를 줘
앞에 있는 단어일수록 검색 노출 우선순위가 높아짐

따라서 전환 가능성이 높은 리치키워드는 무조건 앞쪽에 넣는 것이 효과적

📌 이유 2. 보완 키워드는 유입량을 키우는 장치
노출량은 많지만 클릭률은 낮은 키워드일 경우, 보완 키워드로 중간에 넣어 활용

특히 광고 클릭 유도에 활용 가능 (경쟁강도 낮은 경우 한정)

📌 이유 3. 고정 키워드는 구매 결정 요소
상품 상세 정보가 필요할 땐 반드시 끝부분에 있어야 함

예: "3세트", "1봉", "캡포함" 등

👉 너무 앞에 있으면 검색 유입에 방해됨
