📌 이번 주 도착배송 준비 체크리스트

✅ 1. 풀필먼트 테스트 환경 구축 테스트
- 인어피규어를 통해 도착배송의 프로세스를 테스트
- 이후 어떻게 상품입고 및 테스트 해야할지 결정

✅ 2. 상품선별 및 소량으로 도매처 제품의 입고 테스트 
- 피규어 입고후 판매등 프로세스가 확인되면 도매처 제품을 입고하여 테스트 
- 현재 원스톱, 미니멀, 파타르 각각에 1개의 제품을 선정해서 테스트 
   이과정에서 각각 신청을 따로 해야하는지도 검증 




1️⃣ 풀필먼트 테스트 환경 구축
(1) 네이버 풀필먼트(NFA) 서비스 신청 - '품고' 신청완료 
   - 각각의 상품마다 서비스를 신청해야 하는가?

(2) 풀필먼트 적용 프로세스 정리
기존 3PL과 병행하면서 풀필먼트 적용이 가능한지 검토
도매처에서 풀필먼트 창고로 직접 배송이 가능한지 확인
도매처 → 풀필먼트 창고 → 고객 방식으로 운영 가능 여부 확인
   - 아무래도 될듯, 일단 피규어로 테스트

2️⃣ 도착배송 테스트 상품 선정
(1) 판매 데이터 기반으로 1차 필터링
최근 6개월~1년간 꾸준히 팔린 제품 중 리뷰가 있는 제품 찾기
반품율이 낮고, **객단가(AOV)**가 높은 제품 우선 선정

(2) 1차 테스트용 제품 3~5개 선정
소량(50~100개)만 풀필먼트에 입고하여 초기 테스트 진행
네이버 도착보장 태그 적용 후 판매량 변화 모니터링

3️⃣ 시스템 및 프로세스 설정
도착배송 적용 여부를 상품 데이터(관리코드) 에 추가
상품명 가공 및 이미지 필터링에 ‘도착배송 태그’ 포함
정기적인 재고관리 루틴 필요

4️⃣ 풀필먼트 첫 입고 진행
(1) 풀필먼트 센터에 테스트 상품 소량 입고
50~100개 단위로 소량 입고 후 판매 데이터 분석
물류사 프로세스가 원활히 돌아가는지 체크

(2) 도착배송 테스트 상품의 판매 데이터 모니터링
도착배송 적용 전후 클릭률(CTR) & 전환율(구매율) 변화 체크
광고 노출, 검색 순위, 판매량 증가 여부 분석

5️⃣ 테스트 후 개선 방향 결정
(1) 1주일간 판매 데이터 수집 및 분석
풀필먼트 도입 후 실질적인 판매량 증가가 있었는지 확인
추가 비용(수수료 1,000~1,500원) 대비 수익성이 유지되는지 분석
